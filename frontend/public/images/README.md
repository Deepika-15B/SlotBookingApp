# Background Images Instructions

## Dashboard Background Image

1. Save your pink-to-teal gradient background image as `dashboard-background.jpg` or `dashboard-background.png`
2. Place it in this folder: `frontend/public/images/`
3. The image will automatically be used as the background for all dashboard pages (Admin and User)

## Login/Register Background Image

1. Save your futuristic tech background image as `auth-background.jpg` or `auth-background.png`
2. Place it in this folder: `frontend/public/images/`
3. The image will automatically be used as the background for Login and Register pages

## Supported Formats
- JPG/JPEG
- PNG
- WebP

## Recommended Image Specifications
- Resolution: 1920x1080 or higher
- Format: JPG or PNG
- File size: Under 2MB for optimal performance

## Note
If no image is provided, the CSS will use beautiful gradient fallbacks that match the pastel aesthetic.
